# 🐍 Snake, Water, Gun Game (Python)

A simple **Snake, Water, Gun** game built using **Python**.  
This is a beginner-friendly project that uses basic concepts like:
- Dictionaries
- Conditional statements
- Random module
- User input handling

---

## 🎮 How the Game Works

- Snake 🐍 drinks Water 💧 → **Snake wins**
- Water 💧 drowns Gun 🔫 → **Water wins**
- Gun 🔫 kills Snake 🐍 → **Gun wins**
- Same choice → **Draw**

---

## 🧠 Game Logic

- Computer randomly chooses between:
  - `snake`
  - `water`
  - `gun`
- User inputs:
  - `s` for Snake
  - `w` for Water
  - `g` for Gun
- The program compares both choices and declares the result.

---

## 🛠️ Technologies Used

- **Python 3**
- **random module**
